# Archived Post-Processing Scripts

This directory contains post-processing support scripts that are not part of the public release workflow.

The public post-processing surface is intentionally limited to:

- `scripts/post_processing/relabel_three_phase_generated_labels_batch.py`
- `scripts/post_processing/build_drop_changed_profiles.py`
- `scripts/post_processing/build_closure_bridge_profiles.py`

The archived files are retained for provenance, historical analysis, diagnostics, export preparation, and one-off relabeling utilities.
